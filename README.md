**NeuroFlow** is a browser-based sound generator for deep focus and cognitive immersion.  
It combines **procedural audio synthesis**, **environmental soundscapes**, and an integrated **Pomodoro timer**, all running locally in the browser using the **Web Audio API**.

The project is dependency-free and designed as an experiment in sound design, UX, and cognitive environments.

## How to Use

1. Click **Play** to start the sound engine and focus session  
2. Select a **cognitive profile** to change the soundscape and visual atmosphere  
3. Open the **Mixer** to fine-tune individual sound layers  
4. Adjust the **master volume** as needed  
5. Use the built-in **Pomodoro timer** to alternate between focus and rest phases  

All changes are applied in real time.

## Project Scope

NeuroFlow explores how **procedural sound design**, **minimal UX**, and **real-time audio control** can be combined to create immersive environments for focus and mental flow.

The project is intended as:
- A Web Audio API experiment
- A sound design and interaction study
- A foundation for future expansion (e.g. new profiles, persistence, localization)
